# Playwright Starter Kit (QA Interview Ready)

![Playwright Tests](https://github.com/lastlycan/playwright-starter-kit/actions/workflows/playwright.yml/badge.svg)

A minimal but interview-ready Playwright TypeScript starter showcasing:
- POM structure
- UI + API tests
- CI pipeline with HTML report artifacts

---
## 🚀 Quick Start

```bash
npm install
npx playwright install
npm test
---


## 🚀 What’s included

- UI automation using Playwright
- Page Object Model (POM) structure
- API testing example
- Tagged runs (smoke/regression/api)
- CI: GitHub Actions + HTML report artifact upload
